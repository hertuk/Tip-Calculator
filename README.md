💰 Tip Calculator — Multi‑Language Showcase

Welcome to the **Tip Calculator** repository – a single tool implemented in **7 different programming languages**.  
Whether you're learning a new language, comparing syntax, or just need a reliable tip calculator, this project has you covered.

## ✨ Features
- **Flexible input** – enter any bill amount, tip percentage, and number of people.
- **Preset percentages** – you can type 15, 20, 25, or any custom value.
- **Automatic split** – calculates tip and total per person.
- **Robust validation** – handles invalid inputs gracefully.
- **Clean output** – shows all numbers with two‑decimal precision.

## 🚀 How to Run
Each file is standalone. Use the appropriate interpreter or compiler:

| Language | Command |
|----------|---------|
| Go       | `go run tip_calculator.go` |
| Python   | `python tip_calculator.py` |
| C#       | `dotnet run` (or compile with `csc`) |
| JavaScript | `node tip_calculator.js` |
| Java     | `javac TipCalculator.java && java TipCalculator` |
| Ruby     | `ruby tip_calculator.rb` |
| Swift    | `swift tip_calculator.swift` |
